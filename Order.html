Tips: You can use the POSTMAN to test API

post:/order.json
*long:amount#（In cents. i.e 100= 1RMB, 1000=10RMB）
*bizNo#（APP transaction number produced and kept by client for record keeping.）
*goodsTitle#（APP order title. This will be displayed during payment. i.e "Premium subscription 3 months"）
*payChannel#（target pay channel, i.e. WECHAT or ALIPAY）
customerIdentity#（recommend fill, refer to purchase restore section/ sms code section）
attachData#（optional field for extra data you may need to attach and send to our server）
sourceFrom#（optional field to attach app store names in the event you choose to track payments from each different store. This requires additional code in your app as well as different apks for each store）
